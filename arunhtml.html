Projects

Pick And Place System

Technology used (Control System, Automation, Pneumatic control) - The pick and place system is one of the technology in manufacturing industry and designed to perform pick and place functions. We designed a pick and place system which displaces objects from one place to another. It contains a double acting cylinder with relays for electrical circuit for front and back movement. Dual Solenoid Direction control valves are used for movement in linear direction. We used an arduino controller for this project with a magnetic gripper for displacing magnetic objects.

Versatile Farming System

Technology used (Control System, Automation, Machine Learning) - In this project we are designing a automated mobile robot which is capable of plowing, seeding, fertilizing and irrigating, all at the same time. It also checks for the soil moisture using sensor and irrigates the farm whenever necessary. Thus this project promotes farming and help the farmers in various activities in the field.

Waste Management System

Technology used (Control System and IoT) - In the project we equipped a dumpster with 2 ultrasonic sensors - one for detecting a persons hand and open the lid of the dumpster through an actuator via the arduino controller. The other sensor is used for detecting the level of trash inside the dumpster. Once a certain limit of trash has been reached it send a message to the municipal corporation to clean the dumpster via the GSM module. This project promotes for cleaner ecosystem, regular and easy maintenance of dumpster, avoid overflow of dumpster in the city and convenient route planning for the dump trucks.
